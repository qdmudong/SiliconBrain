# SiliconBrain Assets

This directory contains visual media for the project's GitHub README and documentation.

## Recommended Assets for Maximum Impact:

1.  **`logo.png`**: A stylized 🧠 or "SiliconBrain" icon.
2.  **`knowledge_map_demo.png`**: A high-resolution screenshot of the interactive `brain_map.html` showing the 5,000+ interconnected nodes.
3.  **`dashboard_chat.png`**: A screenshot of the Streamlit Chat interface showing the "Thinking Process" and "Efficiency Report".
4.  **`mastery_loop.gif`**: A short recording of the "Recursive Domain Mastery" loop populating the graph.

## How to add to README:
Use the following syntax in the main `README.md`:
`![Description](assets/filename.png)`
